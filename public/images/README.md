# Image Assets

Place your MSU images in this directory:

## Required Images:

1. **hero.png** - MSU building background image (1920x600px recommended)
   - Should show the Midwestern State University campus building
   - Will be used as hero section background

2. **logos.png** - MSU logo (200x70px recommended)
   - Official Midwestern State University logo
   - Displays in top-left of hero section

3. **msu-footer-logo.png** - MSU footer logo (150x50px)
   - Footer branding image
